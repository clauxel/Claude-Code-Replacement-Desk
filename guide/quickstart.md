# Quickstart

Claude Code Replacement Desk is a SaaS workspace for Claude Code replacement planner.

## Fast Path

1. Open Claude Code Replacement Desk and choose a bounded pilot workflow.
2. Enter public-safe examples for Claude Code replacement planner.
3. Review the generated queue, evidence, or approval state with the owner.
4. Use pricing and checkout links when the pilot needs a paid workspace.
5. Export the resulting report or evidence for the team record.
